# 行動中のエネルギー消費量（Kcal）
## 簡易な指揮 - A式
行動中のエネルギー消費量 = 体重（kg） * 行動時間（h） * 5
## 汎用性の高い式 - B式
行動中のエネルギー消費量 = (1.8 * 行動時間（h）） + （（0.3 * 歩行距離（km）） + （10.0 * 上りの累積標高差（km）） + （0.6 * 下りの累積標高差（km））） * （体重（kg） + 装備の重量（kg）） 
### 時間の要素
1.8 * 行動時間（h）
### 距離の要素
（0.3 * 歩行距離（km）） + （10.0 * 上りの累積標高差（km）） + （0.6 * 下りの累積標高差（km））
### 重さの要素
体重（kg） + 装備の重量（kg）
- 「装備の重量」は衣類、靴、ザックなど身につけているものの重量
## 式の説明
- B式は、荷物が多い場合や、歩行速度が標準コースタイムから外れる場合でも適用できる。
- B式に、軽装かつ標準タイムで歩くときの値を代入すれば、A式とほぼ同じ値が得られる
- A式とB式ともに、kcalをmlと読み替えると脱水量を表す式として使用できる。

# 生活中のエネルギー消費量（kcal）
## 基礎代謝量の式 - C式
生活中のエネルギー消費量 = 体重（kg） * 生活時間（h） * 1時間あたりの基礎代謝基準値
## 式の説明
- 一泊以上の登山では宿泊中のエネルギー消費量を考える必要がある。
- その目安に基礎代謝量を利用する。
- 基礎代謝量は、目覚めた状態で横になっている時の値
- 座っているときにはその1.2倍、寝ているときには0.8倍くらいの値になる
- 実用性を考慮し、山での生活時間の半分を起きている時間、半分を寝ている時間と考え身体活動レベルは1.0とする

# エネルギーの計算例
- 20歳の男性、体重60kgの人が泊まりの登山を行った
- 行動時間が8時間、生活時間が16時間とする
- 行動中のエネルギーはA式で60kg * 8h * 5 = 2,400kcal
- 生活中のエネルギーはC式で60kg * 16h * 1 = 960kcal
- 一日のエネルギーは合計の3360kcal
